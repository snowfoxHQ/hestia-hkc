# Layer: Integrations (NOT Core)

hkc-crystallizer 属于 **HKC Integrations Layer**,不属于 HKC Core。

HKC Core:   hkc-core / hkc-kde / hkc-kee / hkc-ace / hkc-search
Integrations: hkc-crystallizer(边界转换器:Memory/Event World → Knowledge World)

它依赖 HKC(通过 KnowledgeIngress 协议),但不是 HKC 知识系统的本体。
区别对待的意义:Core 的演化(KDE/KEE 重构)不应被集成层牵动,反之亦然。
